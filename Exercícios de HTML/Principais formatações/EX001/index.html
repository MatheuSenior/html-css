Textos de todas as aulas de html

Negrito

Negrito é um estilo de fonte que enfatiza palavras de maneira forte e impactante
ÀS vezes, é usado para chamar a atenção para informações impostantes.

Itálico

Itálico (ou ênfase) é uma forma sutil de destacar palavras, dando-lhes uma inclinação elegante. Pode ser usado para adicionar uma camada de ênfase ao texto.

Texto marcado

texto marcado é frenquetemente usado para indicar algo como um "destaque". É semelhante ao itálico, mas pode ter uma aparência mais destacada, como um marcador colorido em um texto físico.

Texto pequeno

Às vezes, você precisa de texto pequeno para notas de rodapé ou informações adicionais. Isso é especialmente útil quando se deseja fornecer contexto extra.

Texto deletado

Texto deletado é usado para mostrar que uma parte do texto foi removida ou riscada. Isso pode ser útil para indicar mudanças em um documento ou destacar informações desatualizadas.

Texto inserido

Texto inserido é uma maneira de mostrar adicionais a um texto. Pode ser útil em colaborações de redação para destacar as contribuições de diferentes autores.

Texto sobrescrito

Texto sobrescrito é aquele em que os caracteres são colocados acima da linha de base normal. Isso é comum em equações matemáticas, como x2.

Texto subscrito

Texto subscrito é o oposto de sobrescrito, com caracteres abaixo da linha de base. Isso é comum em fórmulas químicas, como H2O
