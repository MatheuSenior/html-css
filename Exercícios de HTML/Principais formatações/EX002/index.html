Principais formatações

Negrito-Italico

Na era digital, a tecnologia desempenha um papel essencial em nossa vida cotidiana. Com a proliferação de dispositivos eletrônicos, a comunicação se tornou mais rápida e acessível.

Texto marcado

Em um mundo de informações rápidas, o destaque é fundamental. Podemos usar o formato de "Texto marcado" para criar ênfase visual, destacando conceitos-chave. Isso é particulamente útil em apresentações e documentos técnicos.

Texto pequeno

Às vezes, precisamos compartilhar informações detalhadas, mas não queremos distrair o leitor. O uso de texto pequeno é uma abordagem eficaz para fornecer contexto adicional sem prejudicar a fluidez do texto principal.

Texto deletado

As revisões em documentos colaborativos são comuns. Quando uma ideia é descartada, usamos texto deletado para indicar a mudança. Isso ajuda os colaboradores a entender as edições feitas ao longo do tempo.

Texto inserido (sublinhado)

Para mostrar contribuições individuais, é valioso usar texto inserido. Por exemplo, em um projeto de redação em equipa, cada membro pode ter sua voz representada, destacada pelo formato de texto inserido.

Texto sobrescrito

Equações cientificas frequentemente exigem elementos sobrescritos, como fórmulas matemáticas com expoentes, como e=mc2.

Texto subscrito

Da mesma forma, em fórmulas químicas, encontramos elementos subscritos, como a fórmula de dióxido de carbono (CO2).
