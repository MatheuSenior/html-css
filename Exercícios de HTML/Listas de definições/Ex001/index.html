Lista de Definições Simples

Exercicio 1: Lista de Definições Simples

HTML
HyperTextMarkup Language - a linguagem de marcação usada para criar páginas da web.
CSS
Cascading Style Sheets - uma linguagem usada para estilizar apresentações de documentos HTML.
JavaScript
Uma linguagem de programação amplamente usada para adicionar interatividade e comportamento a páginas da web.

Exercício 2: Lista de Definições Aninhadas

Ecossistema
Um sistema formado por comunidades bióticas (organismos vivos) e abióticas (fatores não vivos) que interagem em um determinado ambiente.
Floresta Tropical
Um tipo de ecossistema caracterizado por altos níveis de precipitação e biodiversidade. Geralmente é dividido em floresta tropical úmida e floresta tropical sazonal.
Deserto
Um ecossistema árido e seco, caracterizado por baixos níveis de precipitação e temperaturas extremas. Pode ser classificado em desertos quentes e desertos frios.

Exercício 3: Adição de Elementos Adicionais

HTML
HyperText Markup Language - a linguagem de marcação usada para criar páginas da web.
Exemplo: Um exemplo comum de uso do HTML é a criação de páginas da web com títulos, parágrafos e links.
CSS
Cascading Style Sheets - uma linguagem usada para estilizar a apresentação de documentos HTML.
Exemplo: Ao usar CSS, é possivel aplicar cores, fontes e layounts específicos a elementos HTML, tornando a página mais esteticamente agradável.
JavaScript
Uma linguagem de programação amplamente usada para adicionar interatividade e comportamento a páginas da web.
Exemplo: Com JavaScript, é possivel criar formulários interativos, animações e até mesmo comunicar-se com servidores para carregar dados dinamicamente.
